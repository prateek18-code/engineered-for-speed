#  Engineered for Speed

A high-performance **Formula 1 inspired landing page** featuring a cinematic video background and minimal UI design focused on speed and precision.

---

##  Features
-  Dynamic F1 video background  
-  Minimal and clean UI  
-  Speed-inspired design aesthetics  
-  Responsive layout  
-  Focused headline: *"Engineered for Speed"*  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  

---

## 💡 Inspiration
Inspired by the fast-paced world of Formula 1 racing, this project aims to capture the feeling of speed, precision, and performance through UI design. I learned how to use a video as a background
